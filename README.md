# HXConfig
A [Helix](https://github.com/helix-editor/helix) configuration featuring a port of the [moonfly](https://github.com/bluz71/vim-moonfly-colors) theme.

## Installation
HXConfig can be installed with the following command:
```
git clone https://github.com/proxin187/hxconfig $HOME/.config/helix
```
